CSS(Cascading Style Sheet)

This week's topic is CSS.

CSS stands for Cascading Style Sheet and is used to style elements.

Structure of CSS consists of:

  • Selector
  • Property
  • Value

    • Selectors

    Common selectors are element,ID,class and universal.

  • Element selectors-targets all examples of a particular element
  • ID selectors-target elements with a specific ID
  • Class selectors-target elements with a specific class
  • Universal selectors-targets all elements>

    • Properties

    CSS properties include:

  • Colors-styling elements with background and text colors
  • Font-customize text appearance
  • Text alignment-controlling the placement of a text
  • Padding-space around the content of an element
  • Margin-space outside an element
